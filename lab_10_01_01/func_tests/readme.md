# Тесты для задания 1

- Входные данные:
```
./app.exe ВХОДНОЙ_ФАЙЛ_1 ВХОДНОЙ_ФАЙЛ_2 ВЫХОДНОЙ_ФАЙЛ
./app.exe f ЦЕЛ_ЧИСЛО ВХОДНОЙ_ФАЙЛ ВЫХОДНОЙ_ФАЙЛ
```
- Выходные данные - `файл`

## Позитивные тесты

1. Поиск первого элемента
2. Поиск предпредпоследнего элемента
3. Неотсортированный список
4. Отсортированный список
5. Отсортированный в обратном порядке список
6. Единичный список

## Негативные тесты

1. Слишком много аргументов
2. Слишком мало аргументов
3. После ключа `f` не число
4. Невозможно открыть `ВХОДНОЙ_ФАЙЛ_1`
5. Невозможно открыть `ВХОДНОЙ_ФАЙЛ_2`
6. Входной файл пустой
7. Элемент списка – не число
8. В списке нет искомого числа
9. Искомое число последнее в списке
